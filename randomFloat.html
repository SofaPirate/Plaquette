randomFloat()

This function returns a random real-valued number.

Example

#include <Plaquette.h>

DigitalOut led(13);

void begin() {
}

void step() {
  // 2% probability to toggle the LED
  if (randomFloat() < 0.02)
    led.toggle();
}

Reference

float pq::randomFloat()

Generates a uniform random number in the interval [0,max).
